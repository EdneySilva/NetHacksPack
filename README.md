What is the .NET HacksPack?
=====================
A smart set of common classes and implementations distrubuted as little packages to improve your development productivity.

### Why HacksPack?
- .NET HacksPack is being developed in a way to transform it self and your project in a structured "lego application", where you can detach any party from HacksPack/project and replace it by a custom implementation, or another .NET HacksPack that provide the desiered tools.
- Yes you read correctly, as I sad you can also replace part of the .NET HacksPack implemantation. It is helpfull when part of the solution is not satisfing your project's goals but you want implement the inteire comunication with a queue e.g
- And as a "Lego Application" you can change your backstage without any impact for you business logic. "Hack" your project ;D changing the .NET HacksPack implementation.

## Get Started
.NET HacksPack can be installed using the Nuget package manager or the `dotnet` CLI.

```
dotnet add package NetHacks.Core
```
